## Quick Start

1. Get the latest version

```shell
git clone https://github.com/Ijisrael42/wyzebot_server.git WyzeBots
cd WyzeBots
```

2. Run

```shell
npm install
```

3. Run

```shell
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
